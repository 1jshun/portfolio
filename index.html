DnDn

건설 현장 운영 데이터를 연결하는 통합 관리 플랫폼

BE GitHub FE GitHub
Description

DnDn은 건설 현장에서 발생하는 인력, 근태, 안전, 공정, 문서, 환경 데이터를 하나로 연결해 관리하는 통합 플랫폼입니다. 현장 소장과 관리자가 근로자 현황, 게이트 혼잡도, 공정 진행률, 일일 보고, ESG 지표를 한 화면에서 확인하고 데이터 기반으로 의사결정할 수 있도록 설계했습니다.

Contribute
  • Spring Boot + JPA 기반 핵심 도메인 API 설계 및 구현
  • JWT + Spring Security 기반 Stateless 인증·인가 구조 설계
  • Kafka 기반 비동기 처리 구조를 적용하여 API 요청 흐름과 후속 처리 로직 분리
  • Elasticsearch 기반 검색 기능 구현으로 복잡한 조건 조회 로직을 단순화하고 검색 흐름 개선
  • 검색 엔진 장애 상황을 고려한 RDB fallback 처리로 검색 기능의 안정성 보완
  • Jenkins + Kaniko 기반 이미지 빌드 및 배포 자동화 구성
  • Kubernetes Blue-Green 배포 구조를 적용하여 배포 중 서비스 중단 위험 감소
